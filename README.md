# learn-react-router
How to build a Single Page Application (SPA) using React Router, where navigating between pages updates the URL without reloading the browser. It starts with the basics and gradually introduces nested layouts, dynamic routes, navigation hooks, and data passing.

# React Router Learning Project 🚀

This project was built while learning **React Router** with React. It demonstrates the fundamentals of client-side routing, nested layouts, navigation, and modern routing techniques using `createBrowserRouter`.

## 📚 Topics Covered

* React Router DOM setup
* BrowserRouter
* createBrowserRouter
* RouterProvider
* Routes and Route
* Nested Routes
* Layouts (RootLayout & ContactLayout)
* Outlet
* Link Navigation
* Dynamic Routing Basics
* Component-based Routing
* Organized Project Structure

## ▶️ Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd <project-folder>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
